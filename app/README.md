# 网易云音乐 - 音乐备忘录功能

## 功能介绍

音乐备忘录是网易云音乐的一个新功能，允许用户记录听歌感悟、标记歌曲情绪标签，并支持按时间/标签筛选备忘录，同时提供编辑和删除功能。

## 技术实现

### 1. 页面组件
- 新建了 `src/pages/musicMemo/musicMemo.vue` 组件
- 使用 Element UI 组件库构建界面
- 支持响应式布局，适配移动端和 PC 端

### 2. 路由配置
- 在 `src/router/index.js` 中添加了 `/home/music-memo` 路由
- 左侧导航栏 `src/components/leftNav/leftNav.vue` 中添加了入口

### 3. 功能特点
- **手动记录**：支持输入标题、歌曲名、内容和标签
- **情绪标签**：预设了 8 种常用情绪标签（开心、感动、回忆、治愈、励志、伤感、放松、思考）
- **本地存储**：内容实时存储至 localStorage，刷新页面数据不丢失
- **筛选功能**：支持按时间、标签筛选备忘录
- **编辑删除**：支持对备忘录进行编辑和删除操作
- **模拟数据**：首次加载时自动生成模拟数据用于测试

### 4. 响应式设计
- PC 端：网格布局，每行显示 2-3 个备忘录
- 移动端：单列布局，适配手机屏幕

## 功能演示

1. **新建备忘录**：点击右上角“新建备忘录”按钮，输入标题、歌曲名、内容和标签
2. **编辑备忘录**：点击备忘录底部的“编辑”按钮
3. **删除备忘录**：点击备忘录底部的“删除”按钮
4. **筛选备忘录**：选择筛选方式（全部、按时间、按标签）
5. **查看所有备忘录**：选择“全部”筛选方式

## 注意事项

- 所有数据存储在浏览器 localStorage 中，清除浏览器缓存会丢失数据
- 标签可以自定义输入，也可以选择预设标签
- 时间筛选按从新到旧排序
- 标签筛选支持多标签同时选择
